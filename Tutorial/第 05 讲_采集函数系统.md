## 1. 采集函数的角色

### 1.1 在优化循环中的位置

```
每次迭代：
│
├─► GP 训练完成 → 获得 f̂(θ) 和 s(θ)
│
▼
┌─────────────────────┐
│ AcquisitionFunction │  ← 本讲核心
│     .step()         │
└─────────────────────┘
│
├─► 评估所有候选点的 α 分数
│
├─► 选择 top-k 候选点
│
▼
PyBaMM 评估
```

### 1.2 核心设计思想

**采集函数 = EI × W_charge**

```
α(θ) = EI(θ) × W_charge(θ)   (Eq.14)

其中：
  EI(θ)      — 期望改进（探索未知区域）
  W_charge(θ) — 物理加权（聚焦有希望区域）
```

**直观解释**：
- EI 高：GP 预测可能改进当前最优
- W_charge 高：接近物理启发式的最优区域
- 乘积：平衡探索与开发

## 2. 期望改进 (EI) 计算

### 2.1 EI 公式

```
EI(θ) = (f_min - f̂(θ))·Φ(z) + s(θ)·φ(z)   (Eq.15)

其中：
  z = (f_min - f̂(θ)) / s(θ)               (Eq.16)

符号说明：
  f_min    — 当前最优 Tchebycheff 值
  f̂(θ)   — GP 预测均值
  s(θ)    — GP 预测标准差
  Φ(z)    — 标准正态 CDF
  φ(z)    — 标准正态 PDF
```

### 2.2 代码实现

```python
# acquisition.py:539-614
class EICalculator:
    def __init__(self, gp: GPProtocol, xi: float = 0.0):
        self._gp = gp
        self.xi  = float(xi)  # 勘探奖励（可选）

    def compute(
        self,
        X_candidates: np.ndarray,
        f_min: float,
    ) -> Tuple[np.ndarray, np.ndarray, np.ndarray]:
        """批量计算 EI"""

        # Step 1: GP 预测
        mean, std = self._gp.predict(X_candidates)
        std = np.maximum(std, 0.0)  # 数值保护

        # Step 2: 计算改进量
        improvement = f_min - mean - self.xi

        # Step 3: 计算 z 分数
        with np.errstate(divide="ignore", invalid="ignore"):
            z = np.where(std > 1e-12, improvement / std, 0.0)

        # Step 4: EI 公式
        ei = improvement * scipy_norm.cdf(z) + std * scipy_norm.pdf(z)
        ei = np.maximum(ei, 0.0)  # 确保非负

        return ei, mean, std
```

### 2.3 EI 的行为分析

**三种情况**：

| 情况 | 条件 | EI 行为 |
|------|------|--------|
| 预测优 | f̂(θ) << f_min | improvement 大 → EI 高 |
| 不确定 | s(θ) 大 | φ(z)·s(θ) 项主导 → EI 中等 |
| 预测差 | f̂(θ) >> f_min | improvement 负 → EI≈0 |

**图示**：

```
EI
  │
  │      ╱│╲      峰值在 f̂(θ)≈f_min 且 s(θ) 大处
  │     ╱ │ ╲
  │    ╱  │  ╲
  │   ╱   │   ╲
  │  ╱    │    ╲
  │ ╱     │     ╲
  └───────┴───────→ f̂(θ)
         f_min
```

## 3. 物理加权函数 W_charge

### 3.1 W_charge 公式

```
W_charge(θ) = Πⱼ N(θⱼ; μⱼ, σⱼ²)   (Eq.17)

其中：
  N(θⱼ; μⱼ, σⱼ²) = (1/√(2πσⱼ²)) · exp(-(θⱼ-μⱼ)²/(2σⱼ²))

物理含义：
  μⱼ — 搜索中心（认为最优的区域）
  σⱼ — 搜索范围（基于物理敏感度）
```

### 3.2 代码实现

```python
# acquisition.py:621-703
class WChargeCalculator:
    def compute(
        self,
        X_candidates: np.ndarray,   # (m, 3)
        mu:           np.ndarray,   # (3,)
        sigma:        np.ndarray,   # (3,)
    ) -> np.ndarray:
        """批量计算 W_charge"""

        # 数值稳定：log 空间计算
        diff = X_candidates - mu[np.newaxis, :]   # (m, 3)

        # log W = Σⱼ [-½log(2π) - log(σⱼ) - (θⱼ-μⱼ)²/(2σⱼ²)]
        log_w = -0.5 * np.log(2.0 * np.pi) * 3    # 常数项
        log_w = log_w - np.sum(np.log(sigma))     # -Σ log(σⱼ)
        log_w = log_w - 0.5 * np.sum(
            (diff / sigma[np.newaxis, :]) ** 2, axis=1
        )   # (m,)  -Σ (θⱼ-μⱼ)²/(2σⱼ²)

        # 数值稳定：shift by max
        log_w_shifted = log_w - log_w.max()
        wcharge = np.exp(log_w_shifted)

        # 极端情况保护
        total = wcharge.sum()
        if total < 1e-300:
            return np.ones(X_candidates.shape[0]) / X_candidates.shape[0]

        return wcharge
```

### 3.3 W_charge 的行为

```
W_charge
  │
  │         ▲
  │         │ μ处最大
  │        ╱│╲
  │       ╱ │ ╲
  │      ╱  │  ╲
  │     ╱   │   ╲
  │    ╱    │    ╲
  └───┴─────┴─────┴──→ θ
      μ-2σ  μ  μ+2σ

σ越大 → 曲线越平缓（搜索范围广）
σ越小 → 曲线越尖锐（精细搜索）
```

## 4. μ动态漂移追踪

### 4.1 漂移公式

```
α_t = α_max · exp(-t / t_decay_α) + α_min   (Eq.19)

μⱼ^(t+1) = α_t · μⱼ^(t) + (1 - α_t) · θⱼ^best  (Eq.18)
```

**参数默认值**：
```python
alpha_max     = 0.7   # 初始 LLM 信任度
alpha_min     = 0.05  # 最小 LLM 信任度
t_decay_alpha = 60.0  # 衰减时间尺度
```

### 4.2 物理含义

**早期（t≈0）**：
```
α_0 = 0.7 × exp(0) + 0.05 = 0.75

μ^(1) = 0.75 · μ^(0) + 0.25 · θ_best
      ↑ LLM 先验主导，μ缓慢漂移
```

**晚期（t→∞）**：
```
α_∞ = 0.05

μ^(t+1) = 0.05 · μ^(t) + 0.95 · θ_best
        ↑ 数据主导，μ快速跟踪θ_best
```

### 4.3 代码实现

```python
# acquisition.py:235-373
class SearchMuTracker:
    def __init__(
        self,
        param_bounds: Dict[str, Tuple[float, float]],
        alpha_max: float = 0.7,
        alpha_min: float = 0.05,
        t_decay_alpha: float = 60.0,
    ):
        self._bounds = param_bounds
        self.alpha_max = alpha_max
        self.alpha_min = alpha_min
        self.t_decay_alpha = t_decay_alpha
        self._mu = (lo + hi) / 2.0  # 初始为参数空间中心

    def initialize(
        self,
        theta_best: np.ndarray,
        llm_prior: Optional[LLMPriorProtocol] = None,
    ) -> None:
        """初始化 μ"""
        # 优先级：LLM 先验中心 > θ_best
        if llm_prior is not None:
            mu_candidate = llm_prior.get_warmstart_center()
            if mu_candidate is not None:
                self._mu = mu_candidate
                return

        self._mu = theta_best.copy()

    def update(self, theta_best: np.ndarray, t: int) -> None:
        """每迭代更新 μ"""
        alpha = self.get_alpha(t)
        self._mu = alpha * self._mu + (1.0 - alpha) * theta_best

        # 裁剪到参数边界
        self._mu = np.clip(self._mu, self._lo, self._hi)

    def get_alpha(self, t: int) -> float:
        """计算 α_t"""
        return self.alpha_max * np.exp(-t / self.t_decay_alpha) + self.alpha_min
```

## 5. σ敏感度引导追踪

### 5.1 敏感度公式

```
c = κ · maxⱼ |∂Ψ/∂θⱼ|_{θ_best}           (Eq.21)

σⱼ = c / (|∂Ψ/∂θⱼ| + ε_σ)                (Eq.20)
```

**物理含义**：
- |∂Ψ/∂θⱼ| 大 → Ψ 对θⱼ 敏感 → σⱼ 小 → 细粒度搜索
- |∂Ψ/∂θⱼ| 小 → Ψ 对θⱼ 不敏感 → σⱼ 大 → 粗粒度搜索

### 5.2 停滞扩张

```
若停滞（f_min 未改进）：
  σⱼ^(t+1) = σⱼ^(t) · (1 + ρ)            (Eq.22)

其中 ρ = 0.1（扩张 10%）
```

### 5.3 代码实现

```python
# acquisition.py:380-532
class SearchSigmaTracker:
    def __init__(
        self,
        param_bounds: Dict[str, Tuple[float, float]],
        psi_fn: PsiFunction,
        kappa: float = 0.20,
        eps_sigma: float = 0.001,
        rho: float = 0.1,
    ):
        self._psi_fn = psi_fn
        self.kappa = kappa
        self.eps_sigma = eps_sigma
        self.rho = rho
        self._sigma_max = 0.5 * (hi - lo)  # σ上界

    def initialize(self, theta_best: np.ndarray) -> None:
        """初始化 σ"""
        self._sigma = self._compute_sigma_from_grad(theta_best)

    def update(self, theta_best: np.ndarray, stagnated: bool) -> None:
        """每迭代更新 σ"""
        # Step 1: 重算 σ（Eq.20-21）
        sigma_new = self._compute_sigma_from_grad(theta_best)

        # Step 2: 停滞扩张（Eq.22）
        if stagnated:
            sigma_new = sigma_new * (1.0 + self.rho)

        # Step 3: 裁剪
        self._sigma = np.clip(sigma_new, 1e-4, self._sigma_max)

    def _compute_sigma_from_grad(self, theta_best: np.ndarray) -> np.ndarray:
        """核心计算：Eq.20-21"""
        # 获取原始梯度（不归一化）
        grad = self._psi_fn.gradient_raw(theta_best)
        abs_grad = np.abs(grad)

        # Eq.21: c = κ · max|∂Ψ/∂θⱼ|
        c = self.kappa * abs_grad.max()

        # Eq.20: σⱼ = c / (|∂Ψ/∂θⱼ| + ε_σ)
        sigma_raw = c / (abs_grad + self.eps_sigma)

        return sigma_raw
```

### 5.4 数值示例

```python
# 假设梯度
grad = [81.0, 675.0, 108.0]  # ∂Ψ/∂I₁, ∂Ψ/∂SOC₁, ∂Ψ/∂I₂

# 参数
kappa = 0.20
eps_sigma = 0.001

# 计算
c = 0.20 * max(81.0, 675.0, 108.0) = 0.20 * 675.0 = 135.0

sigma_I1   = 135.0 / (81.0 + 0.001) ≈ 1.67
sigma_SOC1 = 135.0 / (675.0 + 0.001) ≈ 0.20
sigma_I2   = 135.0 / (108.0 + 0.001) ≈ 1.25

# 解释：
# SOC1 最敏感 → σ最小（0.20）→ 精细搜索
# I1 较不敏感 → σ较大（1.67）→ 粗粒度搜索
```

## 6. 综合评分与选择

### 6.1 α分数计算

```python
# acquisition.py:734-809
class AcquisitionScorer:
    def score(
        self,
        X_candidates: np.ndarray,
        f_min: float,
        mu: np.ndarray,
        sigma: np.ndarray,
    ) -> Tuple[np.ndarray, np.ndarray, np.ndarray, np.ndarray, np.ndarray]:
        """计算 α = EI × W_charge"""

        # Step 1: 计算 EI
        ei, mean, std = self._ei_calc.compute(X_candidates, f_min)

        # Step 2: 计算 W_charge
        wcharge = self._wcharge_calc.compute(X_candidates, mu, sigma)

        # Step 3: 综合评分
        alpha = ei * wcharge  # Eq.14

        return alpha, ei, wcharge, mean, std
```

### 6.2 Top-k 选择

```python
def select_top_k(
    self,
    X_candidates: np.ndarray,
    alpha: np.ndarray,
    k: int = None,
) -> Tuple[List[np.ndarray], List[int], np.ndarray]:
    """按 α 降序选 top-k"""

    k = k if k is not None else self.n_select

    # 降序排序
    sorted_idx = np.argsort(alpha)[::-1]
    top_idx = sorted_idx[:k]

    selected_thetas = [X_candidates[i].copy() for i in top_idx]
    selected_indices = top_idx.tolist()
    selected_scores = alpha[top_idx]

    return selected_thetas, selected_indices, selected_scores
```

## 7. AcquisitionFunction 门面类

### 7.1 完整接口

```python
# acquisition.py:854-1147
class AcquisitionFunction:
    """采集函数门面类"""

    def __init__(
        self,
        gp: GPProtocol,
        psi_fn: PsiFunction,
        param_bounds: Dict[str, Tuple[float, float]],
        n_select: int = 3,
        # μ参数
        alpha_max: float = 0.7,
        alpha_min: float = 0.05,
        t_decay_alpha: float = 60.0,
        # σ参数
        kappa: float = 0.20,
        eps_sigma: float = 0.001,
        rho: float = 0.1,
        # EI 参数
        xi: float = 0.0,
    ):
        # 子组件实例化
        self._mu_tracker = SearchMuTracker(...)
        self._sigma_tracker = SearchSigmaTracker(...)
        self._ei_calc = EICalculator(gp, xi)
        self._wcharge_calc = WChargeCalculator(param_bounds)
        self._scorer = AcquisitionScorer(...)

    def initialize(self, database: DatabaseProtocol, llm_prior=None) -> None:
        """ warmstart 后初始化"""
        theta_best = database.get_theta_best()
        self._mu_tracker.initialize(theta_best, llm_prior)
        self._sigma_tracker.initialize(theta_best)

    def step(
        self,
        X_candidates: np.ndarray,
        database: DatabaseProtocol,
        t: int,
        w_vec: np.ndarray = None,
    ) -> AcquisitionResult:
        """执行一次采集步骤"""
        # 1. 读取状态
        f_min = database.get_f_min()
        theta_best = database.get_theta_best()
        stagnated = not database.has_improved()

        # 2. 更新 μ和σ
        self._mu_tracker.update(theta_best, t)
        self._sigma_tracker.update(theta_best, stagnated)

        mu = self._mu_tracker.get_mu()
        sigma = self._sigma_tracker.get_sigma()

        # 3. 计算 α并选 top-k
        alpha, ei, wcharge, mean, std = self._scorer.score(
            X_candidates, f_min, mu, sigma
        )
        selected = self._scorer.select_top_k(X_candidates, alpha)

        # 4. 构建结果
        return AcquisitionResult(
            selected_thetas=selected[0],
            selected_indices=selected[1],
            selected_scores=selected[2],
            all_alpha=alpha,
            all_ei=ei,
            all_wcharge=wcharge,
            all_mean=mean,
            all_std=std,
            state=self.get_state(),
            debug={...},
        )

    def get_state(self) -> AcquisitionState:
        """获取当前状态快照"""
        return AcquisitionState(
            mu=self._mu_tracker.get_mu(),
            sigma=self._sigma_tracker.get_sigma(),
            alpha_t=self._mu_tracker.get_alpha(self._current_t),
            stagnation_count=database.get_stagnation_count(),
            t=self._current_t,
            f_min=self._current_f_min,
            theta_best=self._current_theta_best,
            grad_psi_at_best=self._sigma_tracker.get_last_grad(),
        )
```
