# LLMBO-MO 教程系列索引

## 教程概览

本教程系列专为希望**深入理解 LLMBO-MO 框架**的研究者和开发者设计。通过 7 讲系统性课程，你将：

1. 建立对整体架构的清晰认知
2. 深入理解每个核心模块的工作原理
3. 掌握代码阅读和调试技巧
4. 获得扩展和定制框架的能力

** prerequisites **:
- Python 编程基础
- 基本的机器学习知识
- 对优化问题有初步了解

## 关键公式速查

### Tchebycheff 标量化（第 01 讲）
```
f_tch = max_i(wᵢ·f̄ᵢ) + η·Σᵢ(wᵢ·f̄ᵢ)
```

### 物理代理函数（第 03 讲）
```
Ψ(θ) = Q_nom · [I₁·R̄₁·(SOC₁-SOC₀) + I₂·R̄₂·(SOC_end-SOC₁)]
```

### 复合核函数（第 03 讲）
```
k^(t)(θ,θ') = RBF(θ̃,θ̃') + γ · ∇Ψ(θ)ᵀ W^(t) ∇Ψ(θ')
```

### 采集函数（第 05 讲）
```
α(θ) = EI(θ) × W_charge(θ)
```

### μ漂移（第 05 讲）
```
μⱼ^(t+1) = α_t · μⱼ^(t) + (1 - α_t) · θⱼ^best
```

### σ引导（第 05 讲）
```
σⱼ = c / (|∂Ψ/∂θⱼ| + ε_σ)
```
