## 1. 为什么需要物理信息 GP？

### 1.1 标准 GP 的局限

标准高斯过程使用纯数据驱动核函数（如 RBF）：

```
k_RBF(θ,θ') = exp(-‖θ-θ'‖²/(2l²))
```

**问题**：
- 忽略已知的物理规律
- 需要大量数据才能学习基本趋势
- 外推能力差

### 1.2 LLMBO-MO 的创新

LLMBO-MO 引入**物理代理函数**Ψ(θ)，构造复合核：

```
k^(t)(θ,θ') = RBF(θ̃,θ̃') + γ · ∇Ψ(θ)ᵀ W^(t) ∇Ψ(θ')   (Eq.11)
```

**物理含义**：
- RBF 项：捕捉数据中的残差模式
- 物理项：注入欧姆热物理先验
- γ：控制物理先验的权重（退火调度）

## 2. 物理代理函数 Ψ(θ)

### 2.1 物理动机

**核心思想**：充电过程中的欧姆热是性能的关键决定因素。

```
欧姆热功率：P = I²R

累计欧姆热：Ψ = ∫ P dt = ∫ I²R dt
```

### 2.2 两阶段充电的累计欧姆热

```python
# gp_model.py:118-136
"""
Ψ(θ) = I₁² · R̄₁(SOC₁) · t₁ + I₂² · R̄₂(SOC₁) · t₂   (Eq.6)

其中：
  t₁ = (SOC₁ − SOC₀) · Q_nom / I₁                    (Eq.7a)
  t₂ = (SOC_end − SOC₁) · Q_nom / I₂                 (Eq.7b)
"""
```

**化简**（消去 I²/I = I）：

```
Ψ(θ) = I₁² · R̄₁ · [(SOC₁-SOC₀)·Q_nom/I₁] + I₂² · R̄₂ · [(SOC_end-SOC₁)·Q_nom/I₂]
     = Q_nom · [I₁ · R̄₁ · (SOC₁-SOC₀) + I₂ · R̄₂ · (SOC_end-SOC₁)]
```

### 2.3 代码实现

```python
# gp_model.py:186-203
class PsiFunction:
    def evaluate(self, theta: np.ndarray) -> float:
        """Ψ(θ)，θ = [I1, SOC1, I2]"""
        I1, SOC1, I2 = self._unpack(theta)

        # 阶段平均 SOC
        soc1_avg = (self.SOC0 + SOC1) / 2.0
        soc2_avg = (SOC1 + self.SOC_end) / 2.0

        # 阶段平均内阻
        R1 = self._R_bar(soc1_avg)
        R2 = self._R_bar(soc2_avg)

        # 化简后的公式
        psi = self.Q_nom * (
            I1 * R1 * (SOC1 - self.SOC0) +
            I2 * R2 * (self.SOC_end - SOC1)
        )
        return float(psi)
```

**数值示例**：
```python
theta = [5.0, 0.4, 2.5]  # I1=5A, SOC1=0.4, I2=2.5A
# 假设参数
Q_nom = 18000 C
R_int = 0.015 Ω
SOC0 = 0.1, SOC_end = 0.8

# 计算
R1 = R2 = 0.015 Ω (常数内阻)
Ψ = 18000 × [5.0 × 0.015 × (0.4-0.1) + 2.5 × 0.015 × (0.8-0.4)]
  = 18000 × [0.225 + 0.15]
  = 18000 × 0.375
  = 6750 J (焦耳)
```

## 3. 梯度计算 ∇Ψ(θ)

### 3.1 梯度公式推导

**对 I₁ 求导**：
```
∂Ψ/∂I₁ = ∂/∂I₁ [Q_nom · I₁ · R̄₁ · (SOC₁-SOC₀)]
       = R̄₁ · (SOC₁-SOC₀) · Q_nom                     (Eq.8)
```

**对 I₂ 求导**：
```
∂Ψ/∂I₂ = ∂/∂I₂ [Q_nom · I₂ · R̄₂ · (SOC_end-SOC₁)]
       = R̄₂ · (SOC_end-SOC₁) · Q_nom                 (Eq.10)
```

**对 SOC₁ 求导**（使用乘积法则）：
```
∂Ψ/∂SOC₁ = Q_nom · [I₁(R̄₁ + (SOC₁-SOC₀)·dR̄₁/dSOC₁)
                    - I₂(R̄₂ - (SOC_end-SOC₁)·dR̄₂/dSOC₁)]   (Eq.9)
```

**若 R 为常数**（dR̄/dSOC = 0）：
```
∂Ψ/∂SOC₁ = Q_nom · (I₁ - I₂) · R
```

### 3.2 代码实现

```python
# gp_model.py:206-225
def gradient_raw(self, theta: np.ndarray) -> np.ndarray:
    """原始 ∇Ψ(θ)，量纲：J/A 或 J/unit"""
    I1, SOC1, I2 = self._unpack(theta)

    # 阶段平均 SOC
    soc1_avg = (self.SOC0 + SOC1) / 2.0
    soc2_avg = (SOC1 + self.SOC_end) / 2.0

    # 内阻及其导数
    R1  = self._R_bar(soc1_avg)
    R2  = self._R_bar(soc2_avg)
    dR1 = self._dR_bar_dsoc(soc1_avg) * 0.5
    dR2 = self._dR_bar_dsoc(soc2_avg) * 0.5

    # 三个偏导数
    dPsi_dI1 = self.Q_nom * R1 * (SOC1 - self.SOC0)
    dPsi_dSOC1 = self.Q_nom * (
        I1 * (dR1 * (SOC1 - self.SOC0) + R1) -
        I2 * (R2 - dR2 * (self.SOC_end - SOC1))
    )
    dPsi_dI2 = self.Q_nom * R2 * (self.SOC_end - SOC1)

    return np.array([dPsi_dI1, dPsi_dSOC1, dPsi_dI2], dtype=float)
```

### 3.3 梯度归一化

**问题**：原始梯度量级约为 10²-10³ J/A，会淹没 RBF 项（[0,1]）。

**解决方案**：单位向量归一化。

```python
# gp_model.py:227-242
def gradient(self, theta: np.ndarray) -> np.ndarray:
    """∇Ψ(θ)，供 GP 核函数使用（返回单位向量）"""
    g = self.gradient_raw(theta)

    if self.normalize_grad:
        norm = np.linalg.norm(g)
        if norm > 1e-12:
            return g / norm  # 单位向量
        return g
    return g
```

**物理含义**：
- 保留梯度方向（敏感度比例）
- 消除量纲影响
- 使耦合项 γ·ĝᵀWĝ ∈ [0, γ·λ_max(W)]

## 4. 复合核函数

### 4.1 核函数结构

```python
# gp_model.py:412-508
class PhysicsCompositeKernel:
    """
    k^(t)(θ,θ') = RBF(θ̃,θ̃') + γ · ∇Ψ(θ)ᵀ W^(t) ∇Ψ(θ')   (Eq.11)

    其中：
      RBF(θ̃,θ̃') = exp(-‖θ̃-θ̃'‖²/(2l²))
      θ̃ = min-max 归一化到 [0,1]
    """
```

### 4.2 核矩阵计算（向量化）

```python
# gp_model.py:472-508
def kernel_matrix(self, X, Xp, l, gamma, W) -> np.ndarray:
    """K ∈ ℝ^{n×m}，其中 K_ij = k(X[i], Xp[j])"""

    # ── RBF 项（全向量化） ──────────────────────────────
    X_n  = self.normalize(X)     # (n,3)
    Xp_n = self.normalize(Xp)    # (m,3)

    # ‖xᵢ-xⱼ'‖² = ‖xᵢ‖²+‖xⱼ'‖²-2xᵢ·xⱼ'
    sq_n  = np.sum(X_n  ** 2, axis=1, keepdims=True)   # (n,1)
    sq_m  = np.sum(Xp_n ** 2, axis=1, keepdims=True).T # (1,m)
    D2    = sq_n + sq_m - 2.0 * (X_n @ Xp_n.T)
    D2    = np.maximum(D2, 0.0)   # 浮点保护
    K_rbf = np.exp(-D2 / (2.0 * l ** 2))               # (n,m)

    # ── 物理耦合项（逐行计算梯度） ───────────────────────
    if gamma < 1e-12:
        return K_rbf

    G  = np.stack([self.psi_fn.gradient(X[i])  for i in range(n)])   # (n,3)
    Gp = np.stack([self.psi_fn.gradient(Xp[j]) for j in range(m)])   # (m,3)

    # G W Gp^T：(n,3)·(3,3)·(3,m) = (n,m)
    K_phys = (G @ W) @ Gp.T

    return K_rbf + gamma * K_phys
```

**计算复杂度**：
- RBF 项：O(nm) 向量化
- 物理项：O((n+m)×3) 梯度计算 + O(nm) 矩阵乘法

## 5. 耦合矩阵管理

### 5.1 LLM 提供的三个矩阵

```python
# gp_model.py:265-368
class CouplingMatrixManager:
    """
    管理 LLM 提供的三个目标耦合矩阵

    W_time:  充电时间的耦合结构
    W_temp:  温度的耦合结构
    W_aging: 老化的耦合结构
    """
```

**矩阵结构**（3×3 对称 PSD）：
```
        I₁      SOC₁    I₂
I₁   [ w₁₁    w₁₂    w₁₃ ]
SOC₁ [ w₂₁    w₂₂    w₂₃ ]   其中 w_ij = w_ji
I₂   [ w₃₁    w₃₂    w₃₃ ]
```

### 5.2 权重合成（Eq.11'）

```python
# gp_model.py:313-336
def get_W(self, w_vec: np.ndarray) -> np.ndarray:
    """
    W^(t) = (w₁·W_time + w₂·W_temp + w₃·W_aging) / (w₁+w₂+w₃)

    w_vec: 当前 Tchebycheff 权重向量
    """
    w = np.asarray(w_vec, dtype=float).ravel()
    w = np.clip(w, 0.0, None)
    total = w.sum()
    if total < 1e-12:
        return np.eye(3)

    alpha = w / total
    W = alpha[0] * self._W_time + alpha[1] * self._W_temp + alpha[2] * self._W_aging

    return self._clip_psd(W)  # 浮点保护
```

**物理含义**：
- 当前优化重点由 Tchebycheff 权重决定
- 若 w_time 大，则 W^(t) 更接近 W_time
- 凸组合保证 PSD 性质

### 5.3 PSD 投影

```python
# gp_model.py:354-368
@classmethod
def _clip_psd(cls, W: np.ndarray) -> np.ndarray:
    """轻量版：裁剪负特征值（浮点保护）"""
    evals, evecs = np.linalg.eigh(W)
    if evals.min() >= 0.0:
        return W
    evals = np.maximum(evals, 0.0)
    return (evecs * evals) @ evecs.T
```

## 7. GP 预测（Universal Kriging）

### 7.1 训练过程

```python
# gp_model.py:573-647
def fit(self, X, F_tch, w_vec, t) -> "PhysicsGPModel":
    """
    训练 GP：
    1. update_coupling → W^(t), γ(t)
    2. MLE 学习长度尺度 l
    3. 闭合式求 μ̂, σ̂², Cholesky 分解
    """
    self._X = np.atleast_2d(X).copy()
    self._F = np.asarray(F_tch, dtype=float).ravel().copy()
    n = self._X.shape[0]

    # Step 1: 更新 W^(t) 和 γ(t)
    self.update_coupling(w_vec, t)

    # Step 2: MLE 学习 l
    def neg_log_likelihood(log_l: float) -> float:
        l = np.exp(log_l)
        C = self._build_C(l)
        # ... Cholesky 分解和对数似然计算
        return nll

    res = minimize_scalar(neg_log_likelihood, bounds=(...), method="bounded")
    self._l = float(np.exp(res.x))

    # Step 3: 缓存 Cholesky 分解和求解结果
    C = self._build_C(self._l)
    self._C_chol = cho_factor(C)
    # ... 计算 μ̂, σ̂², C⁻¹ 向量
```

### 7.2 预测公式

```python
# gp_model.py:650-690
def predict(self, X_new: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
    """
    Returns
    -------
    mean : (m,)  f̂(θ_new)，Eq.12
    std  : (m,)  s(θ_new) = √s²，Eq.13
    """
    # c_i = k(θ_new, θ^(i))，形状 (m, n)
    C_cross = self.kernel.kernel_matrix(X_new, self._X, self._l, self._gamma, self._W)

    # Eq.12: f̂(θ) = μ̂ + cᵀ C⁻¹ (F−1μ̂)
    mean = self._mu_hat + C_cross @ self._C_inv_F_centered

    # Eq.13: s²(θ) = σ̂²(1 − cᵀC⁻¹c + (1−1ᵀC⁻¹c)²/(1ᵀC⁻¹1))
    Cinv_c = cho_solve(self._C_chol, C_cross.T)
    cT_Cinv_c = np.einsum("mn,nm->m", C_cross, Cinv_c)
    oneT_Cinv_c = self._C_inv_1 @ C_cross.T

    bracket = 1.0 - cT_Cinv_c + (1.0 - oneT_Cinv_c) ** 2 / self._1T_Cinv_1
    var = self._sig2_hat * np.maximum(bracket, 0.0)
    std = np.sqrt(var)

    return mean, std
```
